## 它到底能做什么？

先说最核心的：**一键启动和切换应用**。

设定好之后，按 F1 直接跳到浏览器，按 F2 切到 Slack，按 F3 打开代码编辑器——如果应用已经开着，再按一次会最小化回 Dock。整个切换过程不用鼠标、不用 `⌘Tab` 翻半天，手指还没离开键盘就完成了。

对那些每天要在五六个软件之间来回跳的人来说，这个功能的体感提升非常明显。

**然后是短按 / 长按 / Fn + 按的三重分工。**

Function Key Pro 把每个功能键的操作分成三种模式：
- **短按**：启动或切换指定 App
- **长按**：触发另一个功能，比如截图、静音、打开某个文件夹
- **Fn + 短按**：第三种自定义操作

也就是说，同一个 F 键，不同按法有不同的用途。原来 12 个键能干 12 件事，现在最多能干 36 件事。

**最后是简化复杂快捷键。**
